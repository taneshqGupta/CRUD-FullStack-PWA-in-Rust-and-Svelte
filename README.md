### Task Manager
#### This is a starter CRUD template for Rust and SvelteKit

##### Features:
  ###### Easily installable as a PWA (Progressive Web-Application). 
  ###### Dynamic + Persistent ThemeSwitching with multi-theme setup. 

##### Tech Stack:
  ###### Backend: Rust(Axum)
  ###### Frontend: Sveltekit
  ###### Database: Postgres
